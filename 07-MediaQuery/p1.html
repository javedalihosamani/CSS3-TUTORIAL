MEDIA QUERY

Is used for to create a responsive template.

RESPONSIVE UI

  1. Resolution (width * height) px
  2. Device type (desktop, tablet, mobile)
  3. Orientation (portrait, landscape)
  4. Screen size (small=> Mobile, medium => Tablet, large => Desktop)
  5. Color Depth
  6. Pixel Depth

DEVICES

  • Mobile:
    1. SMALL MOBILE : 320PX - 470PX(SCREEN WIDTH)
    2. LARGE MOBILE : 470PX - 767PX(SCREEN WIDTH)
  • Tablet:
    1. SMALL TABLET : 768PX - 992PX(SCREEN WIDTH)
    2. LARGE TABLET : 993PX - 1024PX(SCREEN WIDTH)
  • Desktop:
    1. SMALL DESKTOP : 1025PX - 1200PX(SCREEN WIDTH)
    2. LARGE DESKTOP : 1201PX - 1400PX(SCREEN WIDTH)
  • 4K SCREEN: 1400PX TO 2560PX
